# EFAR Scheduling Support

This feature covers part-timer availability and scheduling exception handling.

## Included functions

- Part-timer availability records
- Monthly availability view
- WhatsApp contact link
- Half-day coverage gap flag
- Exception filtering and actions
- Bulk exception actions
- CSV export
- Exception audit history
- Browser notification fallback
- Supabase-ready database layer

The frontend is built with React and the backend uses Node.js with Express. Local data storage is used when Supabase credentials are not present.
